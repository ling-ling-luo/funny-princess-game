# 火公主崩铁邀请小程序

这是一个简单的互动网页，展示火公主的图片并询问用户是否想和火公主一起玩崩铁游戏。

## 使用说明

1. 确保 `princess.jpg` 图片文件存在于同一目录中
2. 直接在浏览器中打开 `index.html` 文件
3. 查看火公主图片并回答问题
4. 点击"想"或"不想"按钮查看回应

## 自定义

如果要更换图片，只需将你的图片文件重命名为 `princess.jpg` 并放在同一目录下，或者修改 `index.html` 中的 `src` 属性为你的图片文件名。

## 注意事项

- 这是一个纯前端应用，不需要服务器
- 所有交互都在本地进行，不会发送任何数据
